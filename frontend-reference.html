HTML Semantic

HTML semantic nghĩa là dùng đúng thẻ đúng mục đích. Không phải lúc nào cũng dùng <div>.

Thẻ layout chính

/* Cấu trúc trang đúng chuẩn */
<header>   ← đầu trang, logo, nav
<nav>      ← menu điều hướng
<main>     ← nội dung chính (chỉ 1 cái)
<section>  ← nhóm nội dung có tiêu đề
<article>  ← nội dung độc lập (bài viết)
<aside>    ← sidebar, phụ trợ
<footer>   ← chân trang

Thẻ nội dung

<h1>...<h6>  ← tiêu đề (h1 chỉ 1 lần)
<p>         ← đoạn văn
<ul> <ol>  ← danh sách (unordered/ordered)
<li>        ← phần tử danh sách
<a href=""> ← liên kết
<img src="">← hình ảnh (luôn có alt="")
<button>    ← nút bấm
<input>     ← ô nhập liệu
<form>      ← form nhập liệu

CSS Box Model

Mọi element HTML đều là một cái hộp. Hiểu Box Model = hiểu 80% lý do layout bị vỡ.

MARGIN (khoảng ngoài)
BORDER
PADDING (khoảng trong)
CONTENT (nội dung thực)

Code tương ứng

.box {
  /* MARGIN — đẩy ra ngoài */
  margin: 16px;         ← 4 phía
  margin-top: 8px;     ← chỉ trên

  /* BORDER — đường viền */
  border: 1px solid red;

  /* PADDING — khoảng trong */
  padding: 16px;        ← 4 phía
  padding: 8px 16px;   ← trên/dưới trái/phải

  /* KÍCH THƯỚC */
  width: 300px;
  height: auto;

  /* QUAN TRỌNG: dùng border-box */
  box-sizing: border-box;
}
Chương 03

CSS Flexbox

Flexbox = sắp xếp các phần tử theo 1 chiều (ngang hoặc dọc). Dùng cho navbar, card row, button group...

Bấm để xem flexbox thay đổi

A
BB
CCC
D
justify-content:
align-items:
flex-direction:

Cheat Sheet Flexbox

/* CONTAINER (parent) */
.container {
  display: flex;
  flex-direction: row | column;
  justify-content: flex-start | center | space-between;
  align-items: stretch | center | flex-end;
  flex-wrap: wrap;     ← xuống dòng khi hết chỗ
  gap: 16px;          ← khoảng cách giữa items
}

/* ITEM (child) */
.item {
  flex: 1;             ← chiếm không gian đều nhau
  flex: 1 1 200px;    ← grow shrink basis
  align-self: center; ← override của riêng item đó
}
Chương 04

CSS Grid

Grid = sắp xếp theo 2 chiều (hàng + cột). Dùng cho layout trang, dashboard, gallery...

Grid visual

Cell đầu tiên dùng grid-column: span 2

Cheat Sheet Grid

.container {
  display: grid;
  
  /* 3 cột đều nhau */
  grid-template-columns: repeat(3, 1fr);
  
  /* tự động chia cột ≥ 200px */
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
  
  gap: 16px;
}

.item-wide {
  grid-column: span 2; ← chiếm 2 cột
  grid-row: span 2;    ← chiếm 2 hàng
}

JavaScript — Biến, Hàm, Array, Object

Variables — var / let / const

// const — không thay đổi được (dùng mặc định)
const name = "Minh";
const age = 25;
const isStudent = true;

// let — có thể thay đổi
let count = 0;
count++;   // count = 1
count += 5; // count = 6

// Template string (dùng backtick `)
const message = `Xin chào, ${name}! Bạn ${age} tuổi.`;

Functions — 3 cách viết

// 1. Function declaration
function add(a, b) {
  return a + b;
}

// 2. Arrow function (phổ biến nhất hiện nay)
const multiply = (a, b) => a * b;

// 3. Arrow function nhiều dòng
const greet = (name) => {
  const msg = `Xin chào, ${name}!`;
  return msg;
};

Array — Mảng & các method quan trọng

const fruits = ["🍎", "🍌", "🍊", "🍇"];

// Truy cập phần tử
fruits[0]     // "🍎" (index từ 0)
fruits.length // 4

// Thêm / xóa
fruits.push("🍓")    ← thêm cuối
fruits.pop()         ← xóa cuối
fruits.unshift("🫐") ← thêm đầu

// Duyệt mảng (phổ biến nhất)
fruits.forEach(fruit => console.log(fruit));

// Biến đổi mảng → mảng mới
const upper = fruits.map(f => f.toUpperCase());

// Lọc mảng
const long = fruits.filter(f => f.length > 4);

// Tìm kiếm
const found = fruits.find(f => f.includes("🍎"));

Object — Đối tượng

const user = {
  name: "Minh",
  age: 25,
  skills: ["React", "Node.js"],
  greet: function() {         ← method trong object
    return `Tôi là ${this.name}`;
  }
};

// Truy cập
user.name          // "Minh"
user["age"]        // 25
user.greet()       // "Tôi là Minh"

// Destructuring (tách nhanh)
const { name, age } = user;

// Spread (sao chép + thêm)
const newUser = { ...user, role: "admin" };

JavaScript Events

Event = sự kiện người dùng làm (click, gõ, rê chuột...). JS lắng nghe và phản ứng lại.

Các event phổ biến

// Cách 1: inline (tránh dùng)
<button onclick="doSomething()">Click</button>

// Cách 2: addEventListener (đúng chuẩn)
btn.addEventListener('click', (e) => {
  console.log('Đã click!', e);
});

// Các loại event thường dùng:
'click'      ← bấm chuột
'input'      ← gõ vào input
'change'     ← thay đổi value
'submit'     ← submit form
'keydown'    ← bấm phím
'mouseover'  ← chuột đi vào
'scroll'     ← cuộn trang

DOM Manipulation

DOM = Document Object Model. JS dùng DOM để đọc, thêm, sửa, xóa HTML elements.

Cheat Sheet DOM

// TÌM ELEMENT
const el = document.getElementById('myId');
const el = document.querySelector('.myClass');
const els = document.querySelectorAll('li');

// ĐỌC / THAY ĐỔI NỘI DUNG
el.textContent = "Text mới";   ← chỉ text
el.innerHTML = "<b>Bold</b>"; ← có thể HTML

// CSS / CLASS
el.style.color = "red";
el.classList.add("active");
el.classList.remove("active");
el.classList.toggle("active");

// TẠO / THÊM / XÓA ELEMENT

li.textContent = "Item mới";
list.appendChild(li);
li.remove();

Async / Await & Fetch API

Async/Await = cách gọi API không làm đứng trình duyệt. Hiểu đây là hiểu cách frontend nói chuyện với backend.

Cấu trúc Async/Await

// ❌ CÁCH CŨ — callback hell
fetch('/api/user')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// ✅ CÁCH MỚI — async/await (dễ đọc hơn)
const getUser = async () => {
  try {
    const res = await fetch('/api/user');
    
    if (!res.ok) throw new Error('Lỗi!');
    
    const data = await res.json();
    return data;
    
  } catch (err) {
    console.error('Có lỗi:', err);
  }
};

Responsive Design

Thu nhỏ cửa sổ trình duyệt để thấy layout tự thay đổi.

Trên desktop (2 cột)

Nội dung chính, chiếm 2/3 chiều rộng

/* Mobile First — viết cho mobile trước */
.container {
  display: block;      ← mobile: 1 cột
}

/* Tablet (≥ 768px) */
@media (min-width: 768px) {
  .container { display: grid; grid-template-columns: 2fr 1fr; }
}

/* Desktop (≥ 1024px) */
@media (min-width: 1024px) {
  .container { max-width: 1200px; margin: 0 auto; }
}

Trên mobile (1 cột)

Sidebar tự xuống dưới
